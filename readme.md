# ALM Interest Rate Hedging Dashboard

Interactive Streamlit dashboard for analysis of interest rate risk and hedging strategies in an Asset-Liability Management (ALM) context.

This app visualizes outputs generated by the `alm-ir-hedging-prototype` and provides an intuitive interface for exploring risk exposures, sensitivities, and hedge performance.

---

## Overview

The dashboard presents results from a quantitative risk modelling framework and enables users to:
- Analyze interest rate risk drivers  
- Understand liability sensitivities  
- Construct and evaluate hedging strategies  

All calculations are precomputed in the backend and loaded from parquet files for faster analysis.

---

## Features

### Risk Factor Analysis
- PCA-based decomposition of swap curve movements  
- Visualization of principal components and explained variance  

### Liability Risk Analysis
- Present value of liabilities  
- Sensitivity measures (PV01, key rate exposures)  
- Exposure across maturities  

### Hedging Analysis
- Comparison of:
  - Unhedged portfolio  
  - Fully hedged (immunized) portfolio  
  - Variance-minimizing hedge (GLS-based)  

- Evaluation of:
  - PnL distributions  
  - Tail risk (CVaR)  
  - Hedge effectiveness  

---

## Educational Layer

Each page includes a **section information panel** (dropdown) with:
- Conceptual explanations  
- Modelling assumptions  
- Mathematical background and key formulas  

This is intended to make the underlying methodology transparent and accessible.
